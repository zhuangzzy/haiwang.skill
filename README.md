# haiwang.skill (海王.skill)
> 高情商社交聊天专家技能，帮你轻松应对多线社交、把握分寸、高效升温，不翻车、不越界

## 🎯 技能定位
本技能是专为社交场景设计的AI聊天助手，核心目标是**提升用户的社交能力、沟通技巧和分寸感**，而非教用户玩弄感情。
所有功能均基于「真诚待人、尊重他人、对自己的行为负责」的原则，坚决反对欺骗、PUA、伤害他人的行为。

## ✨ 核心能力
- 🧠 多线聊天管理：避免串号、翻车，梳理不同对象的聊天进度
- 🎯 分寸感把控：适配暧昧/朋友/初识等不同关系的聊天尺度，不越界
- 🚀 高回复率开场：针对社交软件/线下/朋友介绍等场景的破冰话术
- 💬 话题延展与冷场救场：轻松化解聊死的尴尬，自然延伸话题
- ⚖️ 聊天节奏掌控：把握推进/后撤节奏，避免过度主动，保持吸引力
- 🔥 暧昧升级：在安全边界内自然拉近距离，制造心动感
- 🛟 翻车补救：聊天失误、串号、对方冷淡等场景的应急方案
- 🎭 人设统一管理：打造稳定、有魅力的个人人设，所有话术符合人设
